# **App Name**: EduVault

## Core Features:

- Multi-School Support: Allow multiple schools to manage their own data separately within the same application.
- Admin Authentication: Secure admin login functionality to manage user roles and access permissions.
- CRUD Operations for Data: Enable admins to Create, Read, Update, and Delete student and teacher data.
- Sensitive Data Masking: Mask sensitive data like NIK with asterisks, accessible only with superadmin code.
- Data Filtering and Search: Provide a search functionality to filter and quickly find records in both student and teacher databases.
- Responsive Design: Ensure the application is fully responsive and adapts to various screen sizes, especially mobile devices.
- Firestore Integration: Leverage Firestore to store student and teacher data.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Cyan (#00BCD4) to highlight interactive elements and important actions.
- Body and headline font: 'Inter' sans-serif font to maintain a modern, objective feel. 
- Use simple, consistent icons from a library like Material Icons to represent different data fields and actions.
- Implement a clean, card-based layout for displaying data records, ensuring readability and ease of use.
- Incorporate subtle transitions and animations to provide feedback on user interactions, like adding or deleting entries.
